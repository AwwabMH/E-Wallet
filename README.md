The back-end / front-end for an e-wallet application.

Built using:
  - Python / Flask
  - SQLite
  - Stripe (testing visa)
  - Forex API
  - Crypto API
  - HTML/CSS

Includes:
  - SQL injection protection
  - Input checking
  - cuztomizable HTML templates

Features:
  - User transfer
  - Signup/Login system
  - email checking
  - username checking
  - password checking
  - password hashing with salting
  - Real-time converson (Crypto-Forex)
  - Account Top-up with Stripe
